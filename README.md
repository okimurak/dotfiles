# dotfiles_mac

## Requirement

- MacOS
- [prezto](https://github.com/sorin-ionescu/prezto)

## Install

1. Download
   `git clone https://github.com/okimurak/dotfiles_mac.git`

1. Create gitconfig

```config
[user]
  name = <user name>
  email = <email>
[include]
  path = ~/.gitconfig_shared
```

1. Install

```bash
dotfiles_mac/.bin/install.sh
```

1. Enjoy for doffile managements!

## Components

- zsh
- git
